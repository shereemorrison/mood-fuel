# MOOD-FUEL

A bold experimental digital wellbeing campaign website — neo-brutalist, editorial, and intentionally loud.

## Tech Stack

- **Next.js** (App Router)
- **React**
- **Tailwind CSS v4**
- **GSAP** (scroll reveals, marquee, hover motion)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Design

- **Palette:** Orange `#FF5A1F`, Yellow `#FFD400`, Blue `#0057FF`, Cream `#FFF8E7`, Black `#111111`
- **Typography:** Archivo Black (display), DM Sans (body)
- **Style:** Neo-brutalist — thick borders, bright blocks, oversized type

## Build

```bash
npm run build
npm start
```
